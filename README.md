## Phase 1 — Foundations
*~1–2 weeks*
 
- [ ] What is a database? Relational vs. non-relational, tables, rows, columns, primary keys
- [ ] MySQL setup — MySQL Server, MySQL Workbench, CLI, client/server model
- [ ] DDL — `CREATE DATABASE`, `CREATE TABLE`, data types, `DROP`, `ALTER`
- [ ] DML basics — `INSERT`, `UPDATE`, `DELETE`
- [ ] Basic SELECT — `SELECT *`, `WHERE`, `AND/OR/NOT`, `BETWEEN`, `IN`, `LIKE`, `IS NULL`
📁 [`phase-1-foundations/`](./phase-1-foundations/)
 
---
 
## Phase 2 — Querying & Filtering
*~2 weeks*
 
- [ ] Sorting & limiting — `ORDER BY`, `LIMIT`, `OFFSET`, pagination
- [ ] Aggregate functions — `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`
- [ ] `GROUP BY` & `HAVING`
- [ ] Built-in functions — String, Numeric, Date functions
- [ ] Conditional logic — `CASE WHEN`, `IF()`, `IFNULL()`, `COALESCE()`
📁 [`phase-2-querying-filtering/`](./phase-2-querying-filtering/)
 
---
 
## Phase 3 — Joins & Relationships
*~2 weeks*
 
- [ ] Keys & constraints — Primary key, foreign key, `UNIQUE`, `NOT NULL`, `DEFAULT`, `CHECK`
- [ ] Table relationships — One-to-one, one-to-many, many-to-many
- [ ] JOINs — `INNER`, `LEFT`, `RIGHT`, `FULL OUTER`, `SELF`, `CROSS`
- [ ] Subqueries — Scalar, correlated, `EXISTS`, `IN`
- [ ] Set operations — `UNION`, `UNION ALL`, `INTERSECT`, `EXCEPT`
📁 [`phase-3-joins-relationships/`](./phase-3-joins-relationships/)
 
---
 
## Phase 4 — Advanced SQL
*~3 weeks*
 
- [ ] Window functions ⭐ — `ROW_NUMBER`, `RANK`, `DENSE_RANK`, `LEAD`, `LAG`, `SUM OVER`, `PARTITION BY`
- [ ] CTEs — `WITH` clause, chaining CTEs, recursive CTEs
- [ ] Views — Creating/managing views, updatable vs. read-only
- [ ] Indexes — `CREATE INDEX`, composite indexes, full-text indexes
- [ ] Transactions & ACID — `BEGIN`, `COMMIT`, `ROLLBACK`, `SAVEPOINT`, isolation levels
- [ ] Stored procedures & functions — Parameters, `IN/OUT`, user-defined functions
- [ ] Triggers — `BEFORE`/`AFTER` `INSERT`/`UPDATE`/`DELETE`
📁 [`phase-4-advanced-sql/`](./phase-4-advanced-sql/)
 
---
 
## Phase 5 — Database Design & Optimization
*~2 weeks*
 
- [ ] Normalization — 1NF, 2NF, 3NF, BCNF
- [ ] ER diagrams — Entities, attributes, cardinality
- [ ] Query optimization — `EXPLAIN`, `EXPLAIN ANALYZE`, query plans
- [ ] Users, roles & security — `CREATE USER`, `GRANT`, `REVOKE`, SQL injection prevention
- [ ] Backup & restore — `mysqldump`, `mysqlimport`, binary logs
📁 [`phase-5-design-optimization/`](./phase-5-design-optimization/)
